# JARVIS — Dein persönlicher KI-Sprachassistent

JARVIS ist ein kostenloser, lokaler KI-Sprachassistent für Windows im Iron-Man-Stil. Läuft komplett auf deinem PC, keine Cloud-Abhängigkeit außer der KI selbst.

🔗 **Webseite mit allen Infos & Download:** https://lucakroeger350-ctrl.github.io/jarvis-ai-assistant/

## Was JARVIS kann

- 🎤 **Sprachsteuerung per Push-to-Talk** — Mikrofon-Knopf oder globale Tastenkombination (funktioniert auch außerhalb des Fensters)
- 🧠 **Kostenlose KI** — läuft mit Google Gemini (dauerhaft gratis, keine Kreditkarte nötig) oder wahlweise Anthropic Claude
- 🗓️ **Kalender & Erinnerungen** — Termine per Sprache eintragen, vorlesen lassen, automatische Erinnerungen mit Zeitpuffer
- 🖥️ **Bildschirm & Programme** — Screenshots interpretieren ("Was siehst du?"), Programme wie Discord/Spotify öffnen, Dateien öffnen, Websuche
- 📝 **Meeting-Zusammenfassungen** — nimmt Systemaudio auf, transkribiert lokal, erstellt automatisch eine PDF-Zusammenfassung
- 🧩 **Lernfähiges Gedächtnis** — merkt sich Fakten, Vorlieben und beigebrachte Kommandos dauerhaft
- 👥 **Mehrbenutzer-Profile** — eigenes Konto mit E-Mail-Verifizierung pro Person, komplett getrennte Einstellungen
- 🎨 **5 Design-Themes** — Orange, Cyan, Blau, Grün, Lila
- 🔊 **Anpassbare Stimme** — Auswahl, Tonlage, Sprechtempo
- 🚀 **Autostart mit Windows** möglich
- 🔄 **Automatischer Update-Check** — informiert über neue Versionen, Installation nur nach Bestätigung

## Installation

1. Neueste Version unter [Releases](https://github.com/lucakroeger350-ctrl/jarvis-ai-assistant/releases) herunterladen (`JARVIS Setup.exe`)
2. Installer ausführen
3. Beim ersten Start ein kostenloses Konto anlegen (E-Mail wird per Code verifiziert)
4. Kostenlosen Gemini-API-Key auf [aistudio.google.com](https://aistudio.google.com) holen und in den Einstellungen eintragen
