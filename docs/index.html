JARVIS
Dein persönlicher, lokaler KI-Sprachassistent für Windows
Jetzt kostenlos herunterladen Quellcode auf GitHub

Was JARVIS kann

🎤 Sprachsteuerung

Push-to-Talk per Mikrofon-Knopf oder globaler Tastenkombination (auch außerhalb des Fensters) - kein Dauerzuhören nötig.

🧠 Kostenlose KI

Läuft mit Google Gemini (dauerhaft kostenlos, kein Kreditkarte nötig) oder wahlweise Anthropic Claude.

🗓️ Kalender & Erinnerungen

Termine per Sprache eintragen, vorlesen lassen, automatische Erinnerungen inklusive Zeitpuffer zum Losgehen.

🖥️ Bildschirm & Programme

Screenshots interpretieren ("Was siehst du?"), Programme wie Discord/Spotify öffnen, Dateien öffnen, Websuche.

📝 Meeting-Zusammenfassungen

Nimmt Systemaudio auf, transkribiert lokal und erstellt automatisch eine PDF-Zusammenfassung mit Kernpunkten.

🧩 Lernfähig

Merkt sich Fakten, Vorlieben und beigebrachte Kommandos dauerhaft - wird mit der Zeit persönlicher.

👥 Mehrere Profile

Eigenes Konto (E-Mail-Verifizierung) pro Person, komplett getrennte Einstellungen, Erinnerungen und Kalender.

🎨 Individuelles Design

5 Farb-Themes, anpassbare Stimme (Tonlage, Tempo, Auswahl), Autostart mit Windows und mehr.

Installation

1. Auf "Jetzt kostenlos herunterladen" klicken und die neueste JARVIS Setup.exe laden.
2. Installer ausführen (normaler Windows-Installer, keine Kommandozeile nötig).
3. Beim ersten Start ein kostenloses Konto anlegen (E-Mail wird per Code verifiziert).
4. Kostenlosen Gemini-API-Key auf aistudio.google.com holen und in den Einstellungen eintragen.

JARVIS wurde erstellt von Veylo40. Läuft komplett lokal auf deinem PC.
